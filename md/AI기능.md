# Gemini AI 활용 기능 제안

현재 구축된 Guitar Theory 앱(CAGED 시스템, 스케일/코드 시각화)의 특성과 Gemini AI의 능력을 결합했을 때, 사용자에게 실질적인 도움을 줄 수 있는 핵심 기능 제안입니다.

## 1. 문맥 기반 AI 이론 튜터 (Context-Aware Theory Tutor)
**개요**: 사용자가 현재 보고 있는 화면의 내용(Context)을 AI가 인식하고 설명을 제공하는 기능.
단순한 챗봇이 아니라, 앱의 상태를 함께 분석합니다.

*   **기능 상세**:
    *   **"이 스케일은 뭐야?"**: 현재 선택된 스케일(예: C Mixolydian)의 유래, 특징, 자주 쓰이는 장르 설명.
    *   **"이 CAGED 폼은 언제 써?"**: 현재 띄워진 D Form이 실제 연주에서 유리한 상황(예: 특정 코드 진행) 조언.
    *   **인터페이스**: 우측 하단 Floating Button 클릭 시 오버레이 채팅창 출력. 프롬프트 전송 시 `CurrentState JSON`(Key, Mode, Form 등)을 함께 전송.
*   **구현 난이도**: ★★☆☆☆ (기존 상태값 활용 용이)
*   **추천도**: ★★★★★ (가장 기본적이면서 강력한 기능)

## 2. 스마트 스케일 & 코드 매칭 (Smart Recommender)
**개요**: 사용자가 입력한 코드 진행에 어울리는 스케일을 추천하거나, 반대로 스케일에 어울리는 코드를 제안.

*   **기능 상세**:
    *   **Solo Assistant**: "Cm7 - F7 반복 중인데 어떤 스케일 써야 해?" -> "C Dorian을 추천합니다. 특히 6도음(A)을 강조해서 펑키한 느낌을 줘보세요."
    *   **Chord Generator**: "E Phrygian 모드로 작곡하고 싶어. 어울리는 코드 3개만 골라줘." -> "Em, F, G 코드를 활용해 보세요. 특히 F-Em 진행이 Phrygian 특유의 스페니쉬한 느낌을 줍니다."
*   **구현 난이도**: ★★★☆☆ (AI 프롬프트 엔지니어링 중요)

## 3. 맞춤형 연습 루틴 생성기 (Practice Routine Generator)
**개요**: 사용자의 요구사항에 맞춰 체계적인 연습 스케줄(TAB 악보 포함 가능)을 텍스트로 생성.

*   **기능 상세**:
    *   **시나리오**: "나 오늘 스트레칭 좀 하고, CAGED C폼이랑 A폼 연결 연습 20분 하고 싶어."
    *   **AI 응답**:
        1.  **Warm-up (5분)**: 크로매틱 스케일 (TAB: 1-2-3-4...)
        2.  **Technique (15분)**: C Form(3프렛) -> A Form(5프렛) 슬라이딩 연결 연습.
        3.  **Tip**: "손목에 힘을 빼고 이동 시 엄지 위치를 주의하세요."
*   **구현 난이도**: ★★★★☆ (정형화된 포맷으로 응답 유도 필요)

## 4. 작곡/편곡 도우미 (Songwriting Assistant)
**개요**: 음악 이론을 창작에 바로 적용할 수 있도록 돕는 고급 기능.

*   **기능 상세**:
    *   **Modulation 가이드**: "C Major에서 E Major로 전조하고 싶은데 자연스러운 코드는?" -> "B7(Dominant of E)을 피벗 코드로 사용해 보세요."
    *   **Mood Maker**: "몽환적인(Dreamy) 분위기의 아르페지오 패턴 추천해줘."
*   **구현 난이도**: ★★★★☆

## 💡 추천 로드맵

### 1단계: AI 튜터 (Chatbot) 도입
*   **목표**: 사용자가 궁금해하는 이론적 질문 해소.
*   **방법**: 메인 화면 우측 하단에 'AI 질문하기' 버튼 추가.
*   **API 연동**: `SettingsState`에 저장된 API Key를 사용하여 Gemini Pro 모델 호출.

### 2단계: 'Explorer' 탭 기능 확장
*   **목표**: 대화형 코드/스케일 탐색.
*   **방법**: Explorer 탭에 "AI Analysis" 버튼 추가. 현재 선택된 5도권/코드 구성에 대한 심층 분석 텍스트 생성.

### 3단계: 개인화 추천
*   **목표**: 사용자 수준과 취향 반영.
*   **방법**: 연습 기록이나 즐겨찾기 데이터를 기반으로 "오늘의 추천 연습" 제안.
