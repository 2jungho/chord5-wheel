# 작업 절차: 태블릿 프렛보드 크기 조정

## 1. 개요
* 태블릿 환경(화면 너비 600px ~ 1100px)에서 Fretboard Map의 너비가 지나치게 좁게 표시되는 문제 해결.
* 기존의 `0.72` 축소 비율을 데스크탑(1100px 이상)에만 적용하고, 태블릿 구간에서는 화면 너비를 꽉 채우도록 변경.

## 2. 변경 대상 파일
* `lib/widgets/common/fretboard/fretboard_section.dart`

## 3. 상세 작업 내용
1. **코드 분석**: `_buildResponsiveFretboardMap` 메서드 내 너비 계산 로직 확인.
2. **로직 수정**:
    - Mobile (< 600px): 기존 유지 (850.0 고정, 스크롤)
    - Tablet (600px <= width < 1100px): `baseWidth` (100%) 사용
    - Desktop (>= 1100px): `baseWidth * 0.72` (72%) 사용 (기존 로직 유지)

## 4. 검증 계획
* 코드 수정 후 컴파일 에러 확인.
* (가능한 경우) 사용자에게 태블릿 해상도에서의 확인 요청.
