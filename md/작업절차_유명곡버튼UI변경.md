# 작업 절차: 유명곡 버튼 UI 텍스트 포맷 변경

## 1. 개요
* Famous Songs 패널의 노래 버튼 텍스트가 현재 "제목 - 가수" 형태로 한 줄에 표시되어 잘리는 경우가 많음.
* 이를 "제목" (줄바꿈) "(가수)" 형태로 변경하고 가수의 폰트 크기를 줄여 가독성을 높임.

## 2. 변경 대상 파일
* `lib/views/studio/widgets/famous_songs_panel.dart`

## 3. 상세 작업 내용
1. **`_buildSongList` 메서드 수정**:
    * `songs.map` 내부에서 `songTitle` 파싱 로직 추가.
    * ' - ' (공백+하이픈+공백)을 기준으로 제목과 가수를 분리.
    * 분리되지 않는 경우 전체를 제목으로 처리.
2. **UI 레이아웃 변경**:
    * 기존 `Row` > `Expanded` > `Row` 구조를 수정.
    * 텍스트 영역을 `Column`으로 변경하여 제목과 가수를 수직 배치.
    * 가수 텍스트에 괄호 `()` 추가 및 작은 폰트 사이즈 적용.
    * `Container`의 패딩 등 미세 조정 (필요 시).

## 4. 검증 계획
* 문자열 분리 로직이 ' - '가 없는 경우에도 에러 없이 동작하는지 확인.
* UI 오버플로우 발생 여부 확인.
